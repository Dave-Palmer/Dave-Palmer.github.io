Trade Track

Trade Track is my capstone project which was presented to the class at the end of the software engineering course. It's a fullstack web application built with M.E.R.N. It's designed to create a simple way to track tools and items moving between staff in a busy workplace.

Admin User can:
  • -Add/remove individual staff members and set passwords for logging in
  • -Add/remove tools and vehicles and assign them to staff members
  • -Check alerts and reports
Staff Users can:
  • -View all tools and vehicles assigned to them, lend tools to other staff members, report an issue with a tool
  • -Borrow availble tools from the workshop
  • -update Odometer reading on the vehicle assigned to them
